# Governance

ALT-AI is maintained by AffectLog as part of the EDGE-Skills / Prometheus-X initiative.

## Decision Making

- Minor changes (bug fixes, docs) are merged by any maintainer with passing CI.
- Significant changes (new APIs, schema changes, privacy-affecting changes) require review from at least one other maintainer.
- Breaking changes require an ADR under `docs/adr/`.
